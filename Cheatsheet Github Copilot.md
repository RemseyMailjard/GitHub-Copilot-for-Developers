
# GitHub Copilot Slash Commands Cheatsheet

## Available Commands

| Command | Description |
|---------|-------------|
| `/explain` | Explains the selected code in simple terms |
| `/tests` | Generates unit tests for the selected code |
| `/fix` | Proposes a fix for bugs in the selected code |
| `/doc` | Generates documentation for the selected code |
| `/generate` | Generates code to answer your question |

## Usage Tips

- Type `/` in the chat to see all available commands
- Use `/explain` to understand complex code snippets
- Use `/tests` for test-driven development
- Use `/fix` when you encounter errors
- Use `/doc` to auto-document functions and classes
- Use `/generate` for quick code generation from descriptions

## Example

```
/explain
What does this authentication function do?

/tests
Generate tests for the login handler

/fix
Why is this loop throwing an error?
```
